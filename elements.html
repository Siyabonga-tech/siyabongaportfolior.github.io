Climate Change Sentiment analysis

I was involved in creating a Machine Learning model that had the ability to classify whether a person believed in climate change or not, based on their tweet data. The project involved collecting a large dataset of tweets that were related to climate change, and then using various natural language processing (NLP) techniques to preprocess the data and extract useful features. Once the dataset was preprocessed, we used a variety of Machine Learning algorithms such as Logistic Regression, Naive Bayes, and Random Forest to train the model. We also used techniques such as cross-validation and hyperparameter tuning to improve the model's accuracy and ensure that it was not overfitting.

Project Execution

  • Used a tweeter dataset to develop a classification model.
  • Created the API to the model and the App's user interface (UI) with the streamlit framwork.
  • The App takes the string of text and clasifying it as either pro-climate change, anti-climate change or neutral.
  • Hosted the App on Amazon cloud using EC2 instance.
  • Pressented the app and our findings to the stake holders.

    AI OPS Single Pane of Glass

    While I was doing my internship with BCX, I was working in a role of Digital Insights insights and automation. Me and my team we were responsible for Management integration of various sets of data across the AI Ops landscape into consolidated lists to enable technology consolidation and a single view. Single pane of glass is a Unified view of multiple systems or processes in a single interface or dashboard. It provides a single point of control and visibility across all the systems, applications, and infrastructure components in an IT environment, allowing administrators to manage and monitor them from a single location and in a consistent manner. This approach improved operational efficiency, reduced complexity, and enabled faster decision-making by providing a holistic view of the server perfomance.
